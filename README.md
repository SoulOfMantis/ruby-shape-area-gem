# ﻿ruby_shapes

Гем с классами некоторых геометрических фигур и тел, с подсчётом площади и периметра, поверхности и объёма.

### Планируемые фигуры:

1\) Куб, Cube

2\) Прямоугольный параллелепипед, RightCuboid

3\) Сфера, Sphere

4\) Правильная треугольная пирамида/тетраэдр, RegularTetrahedron

5\) Ромбовидная прямая призма, RightRhombicPrism, диагонали основания и высота

6\) Трапециевидная прямая призма, TrapezoidalPrism

7\) Параллелепипед (прямой), RightParallelipiped

8\) прямой эллиптический цилиндр, EllipticalCylinder 

9\) Конус, Cone

10\) Шестиугольная прямая призма, RegularHexagonalPrism
