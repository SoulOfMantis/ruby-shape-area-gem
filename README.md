# ruby-shape-area-gem
Гем с классами фигур и подсчётом их площади.
### Планируемые фигуры:
1. Квадрат, square
2. ​Прямоугольник, rectangle
3. ​Круг, circle
4. ​Треугольник, triangle
5. Ромб, rhombus
6. ​Трапеция, trapezoid
7. Параллелограмм, parallelogram
8. ​Эллипс, ellipse
9. Сектор круга, sector
10. ​Правильный шестиугольник, hexagon
